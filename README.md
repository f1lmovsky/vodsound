# vodsound

RU

Расширение для браузера, позволяющее слышать полноценную звуковую дорожку как на трансляции, без вырезанного аудио.

EN

This browser extension allows you to hear complete audio information, just like in a live broadcast, without any sound being cut out.


Установка:

Скачайте папку "extension"

Откройте ваш браузер и перейдите во вкладку "Расширения"

Включите "Режим разработчика"

Перетяните ПАПКУ в окно браузера, на вкладке расширения ИЛИ нажмите "загрузить распакованное расширение" (эти действия актуальны для Chrome, для вашего браузера может не сработать)

Откройте в браузере https://89.111.155.235 , браузер скажет что подключение не безопасно - нажмите "открыть в любом случае".

Откройте панель расширения, вставьте в строку "Адрес сервера" - https://89.111.155.235.

Готово, расширение установлено.
